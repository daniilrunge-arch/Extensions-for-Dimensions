Ping Extension for proxy for Terraria Tshock Dimensions (Node.js)
A simple extension for the Dimensions proxy server that allows players to check their latency (ping) between servers.
Installation & Setup
To install the extension, follow these steps:
Navigate to your proxy server directory:
Plaintext
\dimensions\build\extensions\
Create a new folder named Ping inside it. The full path should look like this:
Plaintext
\dimensions\build\extensions\Ping
Place the following extension files into the Ping directory:
config.cjs
index.cjs
pinginfo.cjs
Restart your Dimensions proxy to apply the changes.
In-Game Commands
Once installed, players can use the following command:
/ping — Displays your current latency (ping) between servers.
